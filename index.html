Туторіал — AutoMod: швидко й без лишнього драматизму

Короткі пояснення, приклади правил і готові фрагменти для вставки — щоб ваш сервер був охайніший, ніж більшість інтернетів.

Що таке AutoMod Keyword Filters Spam Filters Блокування в нікнеймах Приклади правил

1. Що таке AutoMod

AutoMod — вбудований набір фільтрів для автоматичної модерації повідомлень: ключові слова, спам, масові згадки та ін. Працює по всіх текстових поверхнях (канали, треди, текст у голосових).

Порада: поєднуйте готові списки (Commonly Flagged Words) і власні правила — так у вас буде баланс між автоматизацією і культурою спільноти.

2. Keyword Filters — як налаштувати

  1. Server Settings → AutoMod
  2. Обрати Commonly Flagged Words або Custom Keyword Rule → Set up
  3. Додати ключові слова/фрази або ексепти (exempt words)
  4. Вибрати дію: Block message, Send alert, Timeout user (для кастомних правил)

Wildcards — додавайте `*` на початку/наприкінці слова для префікс/суфікс/вбудованого пошуку (наприклад: `spam*`, `*bot`, `*ana*`).

3. Spam Filters

Два основні режими: блокування спам-контенту (машинне навчання) і обмеження кількості згадок в повідомленні. Машинний фільтр неідеальний — повідомляйте про помилки, якщо бачите фальшпозитиви.

  • Block Spam Content — захищає від реклами та шахрайських посилань
  • Block Mention Spam — ставите ліміт згадок (max 1–50)

4. Блокування слів у нікнеймах

Можна заборонити певні слова в server nicknames або username — користувачі повинні змінити нік, щоб писати в чаті.

  1. Server Settings → Safety Setup → Edit AutoMod → Block Words in Member Profile Names
  2. Додайте слова/фрази — якщо користувач не може змінити нік, йому доведеться змінити основне ім'я

5. Готові приклади правил (скопіюй → встав)

Приклад 1 — блокувати рекламу Nitro / безплатні плюшки (використовувати як Custom Keyword Rule)
free nitro
free nitro code
get nitro free
nitro giveaway
Приклад 2 — wildcard для різних форм слова (наприклад, лайливі варіанти)
*****
sh!t*
badword*

Підказка: оберіть канал для сповіщень AutoMod, де тільки модератори бачитимуть попередження. Не робіть цей канал загальнодоступним, інакше нічого не допоможе.

6. Найпоширеніші помилки

  • Не додають ексепти для нормальних слів — отримують фальшблоки.
  • Налаштування alert-каналу для всіх — модератори отримують тонни повідомлень.
  • Очікують ідеального ML — фільтр вчиться, тому ваша фідбек-крафт потрібен.

Швидкий чекліст

  1. Community-enabled сервер
  2. Зайти: Server Settings → AutoMod
  3. Включити Commonly Flagged + додати Custom Rules
  4. Налаштувати канал для сповіщень (модератори)
  5. Переглянути й оновлювати список + надсилати звіти про помилки

Корисні поради

  • Почни з щадного блокування, протестуй на невеликій групі.
  • Додай role-exempts для ботів і довірених модераторів.
  • Фільтри на нікнейми — радикальний крок, використовуйте обережно.
